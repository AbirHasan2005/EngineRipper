## Setup & Start:
```bash
git clone https://github.com/AbirHasan2005/ToolsEngine
cd ToolsEngine
chmod +x *
./engrip.sh
```

```bash
git clone https://github.com/AbirHasan2005/ToolsEngine && cd ToolsEngine && chmod +x * && ./engrip.sh
```
