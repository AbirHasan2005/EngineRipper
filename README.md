## Setup & Start:
```bash
git clone https://github.com/AbirHasan2005/EngineRipper
cd EngineRipper
chmod +x *
./engrip.sh
```

One line command:
```bash
git clone https://github.com/AbirHasan2005/EngineRipper && cd EngineRipper && chmod +x * && ./engrip.sh
```
